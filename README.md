# KataQuiz
A simple quiz application built with HTML, CSS, and JavaScript.  
Users answer multiple-choice or true/false questions and get a score at the end.  
Users are given a maximum of 15 seconds per question.  

## Features
- Displays questions with multiple-choice or true/false options  
- Tracks score and number of questions  
- Shows results at the end of the quiz  
- Simple UI  

## How to Run
1. Clone the repository  
2. Open `index.html` in any browser  
3. The quiz will load automatically  

## Files
- `index.html` — main page  
- `style.css` — styles  
- `main.js` — quiz logic
